# НОК (наименьшее общее кратное)

```cpp
int lcd(int a, int b) {
    return a / gcd(a, b) * b;
}
```

Нахождение НОК базируется на [алгоритме Евклида](https://ru.wikipedia.org/wiki/Алгоритм_Евклида)
и использует следующую формулу:

$$ a * b \over gcd(a, b) $$
